# TDSControl-examples
A collection of example problems for the analysis and controller design of linear time-invariant time-delay systems.

## Examples:
### Stability analysis
 - **VerheydenEtAl2008**: *K. Verheyden , T. Luzyanina, AND D. Roose. Efficient computation of characteristic roots of delay differential equations using LMS methods. Journal of Computational and Applied Mathematics, 214 (2008)*
 - **WuMichiels2012**: *Wu, Zhen, and Wim Michiels. "Reliably computing all characteristic roots of delay differential equations in a given right half plane using a spectral method." Journal of Computational and Applied Mathematics 236.9 (2012): 2499-2514.*
